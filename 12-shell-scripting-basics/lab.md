# Tasks

1. Create a script at `/tmp/lpic101-report.sh` with a proper shebang.
2. Store the current date in a variable and print it.
3. Accept one positional argument and print it back to the user.
4. Add an `if` statement that checks whether a file exists.
5. Add a `for` loop that prints the contents of a small list.
6. Add a `case` statement with at least two options.
7. Print the script exit code with `echo $?` after running it.
8. Make the script executable and run it directly.
9. Redirect the script output into a report file.
10. Add comments only where they help explain logic.

## Helpful Commands

`#!/bin/bash`, `chmod +x`, variables, `$1`, `if`, `for`, `case`, `exit`, redirection

## Challenge

Extend the script so it prints a mini system report using commands like `hostname`, `uptime`, and `df -h`.
