# Tasks

1. Print the current working directory.
2. List files in the current directory using both normal and long format.
3. Show hidden files.
4. Move to `/tmp`, then return to your home directory.
5. Display your username, effective user ID, and shell.
6. Show the current date and time.
7. Open the manual page for `ls`.
8. Use `whatis`, `type`, and `which` on `cd`, `ls`, and `echo`.
9. Run a few commands and inspect the shell history.
10. Clear the screen and print a custom message like `LPIC-101 lab 01 ready`.

## Helpful Commands

`pwd`, `ls`, `ls -la`, `cd`, `whoami`, `id`, `echo $SHELL`, `date`, `man`, `whatis`, `type`, `which`, `history`, `clear`

## Challenge

Use one command to return to the previous directory after changing directories twice.
