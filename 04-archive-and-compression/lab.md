# Tasks

1. Create a practice directory `/tmp/lpic101-archive` with several sample files.
2. Build a `tar` archive of the directory.
3. List the contents of the archive without extracting it.
4. Extract the archive into a different target directory.
5. Create compressed versions with `gzip`, `bzip2`, and `xz`.
6. Compare file sizes with `ls -lh`.
7. Decompress each file back to its original form.
8. Create a `.tar.gz` archive in a single command.
9. If `zip` is available, create a zip archive and test it.
10. Remove extracted content and restore it again from the compressed archive.

## Helpful Commands

`tar`, `gzip`, `gunzip`, `bzip2`, `bunzip2`, `xz`, `unxz`, `zip`, `unzip`, `ls -lh`

## Challenge

Use `tar` with verbose output to watch files being packed or extracted.
