# LPIC-101 Killercoda Labs

This repository contains a ready-to-use Killercoda course structure for LPIC-101 practice labs.

## Layout

- `structure.json` defines the course order.
- Each `NN-lab-name/` directory is a Killercoda scenario.
- Every scenario uses Ubuntu as the backend image.

## Included Labs

1. Environment and Shell Basics
2. Files and Directories
3. Text Processing Essentials
4. Archive and Compression
5. File Permissions and Ownership
6. Process Management
7. Package Management
8. Boot Process and systemd
9. Hardware and System Architecture
10. Partitions and Filesystems
11. Swap and Filesystem Maintenance
12. Shell Scripting Basics

## Notes

- The scenarios are intentionally vendor-neutral and map to common LPIC-101 objectives.
- The lab instructions are written so learners can complete them manually in the Killercoda terminal.
- You can add `verify.sh`, `setup.sh`, or extra step files later if you want stricter progression.
