# Tasks

1. Create a directory tree under `/tmp/lpic101-files` with subdirectories named `docs`, `bin`, and `archive`.
2. Create three empty files in `docs`.
3. Copy one file into `archive`.
4. Move one file from `docs` to `bin`.
5. Rename one file in place.
6. Use `stat` to inspect one file.
7. Create a hard link and a symbolic link to the same file.
8. Use `find` to list only regular files under `/tmp/lpic101-files`.
9. Remove one file and one empty directory.
10. Display the final tree structure.

## Helpful Commands

`mkdir`, `touch`, `cp`, `mv`, `rm`, `rmdir`, `find`, `stat`, `ln`, `tree`

## Challenge

Find all symbolic links under `/tmp/lpic101-files` using a single `find` command.
