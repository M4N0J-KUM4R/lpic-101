# Tasks

1. Start a long-running command such as `sleep 300`.
2. View the process using `ps`.
3. Start another long-running command in the background.
4. Use `jobs` to inspect background jobs.
5. Bring a background job to the foreground, then suspend it with `Ctrl+Z`.
6. Resume a stopped job in the background.
7. Use `kill` to terminate a specific process by PID.
8. Use `pkill` or `killall` on a process name if available.
9. Inspect system activity with `top` or `ps aux`.
10. Start a command with `nice` and compare its niceness value.

## Helpful Commands

`ps`, `top`, `jobs`, `bg`, `fg`, `kill`, `pkill`, `killall`, `nice`, `nohup`

## Challenge

Launch a command with `nohup` so it continues after the shell exits.
