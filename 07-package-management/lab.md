# Tasks

1. Identify whether the system is Debian-based or RPM-based.
2. On Debian-style systems, run `apt update` and search for a package.
3. Show package information with `apt show` or `apt-cache policy`.
4. List installed packages with `dpkg -l`.
5. Inspect which package owns a file using the distro-appropriate tool.
6. If safe in the environment, install a small package such as `tree` or `htop`.
7. Remove that package using the proper command.
8. Compare high-level package tools like `apt` or `dnf` with low-level tools like `dpkg` or `rpm`.
9. On RPM-based systems, practice `rpm -qa`, `dnf search`, and `rpm -qf`.
10. Record which commands differ by distro family.

## Helpful Commands

`apt`, `apt-cache`, `dpkg`, `dnf`, `yum`, `rpm`

## Challenge

Find the package that provides a common binary such as `ls` or `bash`.
